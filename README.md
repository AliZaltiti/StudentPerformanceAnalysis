# Student Performance Analysis

This repository contains a Python project analyzing student performance data.

## Project: Student Performance Analysis

**Files:**
- `student_data.csv` — dataset containing student grades and attributes
- `student_performance_analysis.py` — Python script performing analysis and visualization

**Libraries Used:**
- Pandas
- NumPy
- Matplotlib

**Tasks Performed:**
- Data inspection and basic statistics
- Grouping and aggregating data (e.g., average grade by gender)
- Scatter plots and bar charts for visualization
- Correlation analysis between parental education and student grades
